# 前端 API 覆盖说明

本前端已封装并接入以下外部接口模块：

- 认证与用户：登录、注册、Token 校验、个人资料、密码修改、管理员用户管理
- 作品：分页、搜索、详情、CRUD、分类、作品分类关联、封面上传
- 角色：分页、搜索、详情、CRUD、角色关系、作品角色演员关联、头像上传
- 人物：分页、搜索、详情、CRUD、人物作品关联、头像上传
- 评分与投票：评分、平均分、个人评分、删除评分、投票与结果、管理员删除评分
- 评论：发表、回复、点赞、编辑、删除、审核、置顶、精华
- 收藏：收藏夹、收藏项、公开收藏夹、管理员删除收藏夹
- 通知：用户通知、未读数、已读、删除、定向发送、系统广播
- 聊天：私聊、群聊、作品群、成员、消息、已读、退出、管理员治理

`/internal/**` 为微服务内部 Feign 接口，前端不应调用。
